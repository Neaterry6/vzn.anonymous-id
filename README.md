# VZN Anon (React + TypeScript + Node.js runtime)

This project is built with **React + TypeScript** using **Vite** (Node.js-based tooling).

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:5173`.

## Pages

- `/` landing page
- `/signup` sign up page
- `/login` login page
- `/feed` post-login feed page
- `/profile` profile settings page
- `/inbox` anonymous inbox page
- `/dashboard` weekly user stats dashboard
- `/group-chat` global anonymous group chat with image + stickers + emoji support
